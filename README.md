# JWL-Assistant
Small but powerful app to Auto toggle Zoom Camera and control OBS functions and OSC Sound Mixer
<img width="632" height="738" alt="image" src="https://github.com/user-attachments/assets/b4cbb42e-ff53-4181-953c-d0f46ee94dba" />
JWL+OBS Assistant is a Windows tool that helps automate live meeting production with OBS and Zoom. It combines scene control, OCR-based switching, Zoom meeting helpers, audio utilities, and quick operator actions in one interface to reduce manual steps during meetings. The goal is simple: make transitions smoother, setup faster, and operation more reliable for congregation AV workflows.
